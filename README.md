# MEAN-STACK

MEAN STACK demo application which help you to easily start your project development with MEAN Stack technology.

# Versions

1. [M] Mongo DB >= 4.x
1. [E] Expressjs >= 4.x
1. [A] Angular >= 8.x.x
1. [N] Nodejs >= 10.x.x

# Prerequisite

1. Node JS >= 10.x.x
2. Mongo DB >= 4.x
3. Angular CLI >= 8.3.x, Install it using  `npm install -g @angular/cli@8.3.20`

# Install Packages

## Client

```sh
cd client
npm i
```

## Server

```sh
cd server
npm i
```

# Start Applications

## Client

```sh
npm start
```

## Server

```sh
npm start
```

## Deploy on Heroku

1. 

