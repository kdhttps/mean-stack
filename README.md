# MEAN-STACK

MEAN STACK demo application which help you to easily start your project development with MEAN Stack technology.

# Versions

1. [M] Mongo DB >= 4.x
1. [E] Expressjs >= 4.x
1. [A] Angular >= 8.x.x
1. [N] Nodejs >= 10.x.x

# Prerequisite

1. Node JS >= 10.x.x
2. Mongo DB >= 4.x

# Install Packages

## Client

```sh
cd client
npm i
```

## Server

```sh
cd server
npm i
```

# Start

## Client

```sh
npm start
```

## Server

```sh
npm start
```
